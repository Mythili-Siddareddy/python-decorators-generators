# Python Decorators and Generators

## Overview

This project demonstrates Python decorators and generators with simple examples and explanation.

## Concepts Covered

* Function decorators
* Wrapper functions
* Generator functions
* Yield keyword

## Project Files

* examples/decorators.py
* examples/generators.py
* docs/report.pdf

## Description

The project includes basic programs to understand how decorators modify functions and how generators produce values efficiently using `yield`.
